# Распознавание смешанного русско-английского языка

## Текущая модель (vosk-model-ru-0.42)

Ваша текущая модель **vosk-model-ru-0.42** уже поддерживает:
- ✅ Русский язык
- ✅ Английские слова в русской речи
- ✅ Технические термины на английском

Модель автоматически распознает английские слова, когда вы их произносите.

## Альтернатива: Многоязычная модель

Для еще лучшего качества смешанного языка можно использовать:

### vosk-model-small-0.22 (многоязычная, ~50 MB)
```bash
# Скачать
axel -n 8 https://alphacephei.com/vosk/models/vosk-model-small-0.22.zip

# Распаковать
tar -xf vosk-model-small-0.22.zip

# Переименовать
move vosk-model-small-0.22 model
```

Поддерживает: английский, немецкий, французский, испанский, русский, турецкий, вьетнамский, итальянский, голландский, каталанский, арабский, греческий, фарси, финский, филиппинский, украинский, казахский

## Использование с Google Speech-to-Text

Для идеального распознавания смешанного языка можно переключиться на Google:

1. Откройте `config.yaml`
2. Измените `engine: "vosk"` на `engine: "google"`
3. Google автоматически определяет язык (настроено в config.yaml)

Google Speech-to-Text лучше справляется со смешанным языком, но требует интернет и API ключ.
